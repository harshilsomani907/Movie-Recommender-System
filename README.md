# Movie Recommender System

This project is a Movie Recommendation System built using Python and Streamlit.  
It recommends movies similar to the selected movie using a similarity matrix.

## Features
- Select a movie from the dropdown
- Get recommended similar movies
- Interactive UI using Streamlit

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## Project Structure

app.py – Streamlit application  
movies.pkl – Movie dataset  
movies_dict.pkl – Movie dictionary  
similarity.pkl – Similarity matrix  
requirements.txt – Required libraries
